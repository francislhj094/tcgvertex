## 🔄 **HOW LANGUAGE DETECTION WORKS**

### **Automatic Detection:**

When a French visitor lands on your site:
1. Browser detects language: `fr`, `fr-FR`, `fr-CA`, `fr-BE`, etc.
2. System automatically sets:
   - Language: French
   - Currency: EUR (€)
   - Locale: `fr-FR`
   - All text displays in French
3. Prices converted to EUR

### **Manual Selection:**
Users can also manually switch countries/languages (if you add a language selector UI later).

## 📚 **DOCUMENTATION**

Translation keys follow this structure:
```javascript
t('home.hero.title')          // Homepage hero title
t('nav.market')               // Navigation "Market" link
t('premium.features.title')   // Premium features section
```

To add new translations:
1. Add to `src/translations/en.js`
2. Add to `src/translations/fr.js` (French version)
3. Add to `src/translations/de.js` (German version)
4. Use with `t('your.key.here')`
